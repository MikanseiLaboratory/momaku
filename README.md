# momaku(網膜)

Cross-platform Desktop app that sends web pages to NDI. 

Inspired by [Vingester](https://github.com/rse/vingester).

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
